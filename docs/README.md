## Documentation

This folder is intended for research documentation, experiment logs, and reports.

Suggested contents:

- **data_description.md** – description of the original Excel format and any domain-specific notes.  
- **experiments.md** – training configurations, hyperparameters, and results tables.  
- **model_comparison.md** – comparison of RandomForest, XGBoost, and MLP regressors using MAE, RMSE, and R².  
- **pipeline_diagram.md** – high-level overview of the data pipeline, model pipeline, and system architecture.
